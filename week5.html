Summary of the week
We found it interesting to observe the behaviour of daily things and interaction between objects and human beings. We are going to make things seem to be automonous by using some force (e.g. magnetic force).

First Prototype


System
diagram1
At the first stage, we give magnetic force to daily objects or define magnetic force inside objects according to the materialities and properties which objects have. And then we interact with them and they give some outputs, such as motion or sound. We expect feed-back cycle happening inside us and daily objects to react as if they have lives.

We hang a set of daily objects on the structure, such as cup, toothbrush, balloon and hammer, which all have different magnitude of electric properties (e.g. as shown in the diagram 5, the toothbrush has +1.2 points, the balloon has +0.5 points and the hammer has -0.7 points). Each of these values represents how much the object pulls another or pushes another. The values are detected by some means and amplified. We move objects according to the amplified values by using servo (diagram 6). And also electric properties in our bodies are sensed and used to interact with the objects (as shown in diagram 7). Each objects has own range of motion(diagram 4).

References

.SHOES OR project
There are several pairs of shoes with motors inside. Motors reacting to human beings and the music enable shoes to look like having lives. People can learn when things move and what triggers motion through experiences, which is a feed-back cycle.

.Skin Care project
When a object is recorded by 3D scanning, it is decomposed into surface texture data and form mesh data and stored. 3D data can only record representations of surface and form, so no matter how precisely recorded 3D data, the contents are empty, unlike real 3D objects. As the Western ghost is often drawn as the existence of only a cloth without body, 3D data also exists like a ghost of skin alone.
We found it interesting that he used daily objects intentionally to depict the distortion or the conflict between virtual and real world.
